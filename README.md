# EnvCheck
+ A tool for automatically detecting server environment information

## Function list
+ Get HostName
  - Print host name
  - Check whether the host name meets the requirements (Only lowercase letters, numbers , < . > < - >)
+ Get CPU
  - Architecture
  - Number of CPU cores
  - Whether the instruction set supports (avx2, AVX, bmi2) 
+ Get GPU
+ Get OS
